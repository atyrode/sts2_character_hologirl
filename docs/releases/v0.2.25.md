# Hologirl v0.2.25

- Applies the current preferred character-select tuning values as defaults:
  - X `842`
  - Y `120`
  - Scale `1.49`
  - Whip density `1.05`
  - Drift density `0.29`
  - Gold jitter `80`
- Adds five temporary background variants to the character-select tuner:
  - Digital Night
  - Backstage Teal
  - Prism Broadcast
  - Muted Encore
  - Void Spotlight
- Persists the selected background variant while switching characters.
- Documents the vanilla-style background direction: broad two-tone fields, simple geometry, quiet negative space, sparse cyan/gold accents, and no detailed illustrated stage.

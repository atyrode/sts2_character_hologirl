# Hologirl v0.2.42

- Adds F3 tuner sliders for character-select hologram shader values:
  - `Holo tint`
  - `Tear`
  - `Shimmer`
  - `Scanline`
  - `Scan speed`
  - `Scan spacing`
- Adds a procedural scanline pass to the character cutout shader.
- Reduces the default tearing applied when selecting hologram character variants.
- Includes shader values in the tuner clipboard output so exact preferred settings can be reported back.

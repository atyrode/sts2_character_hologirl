# Hologirl v0.3.20

- Added cosmetic-only Livestream chat triggers for damage taken, debuffs received, monster deaths, Hologirl death, turn start/end, and generic card play/draw/exhaust moments.
- Added occasional `@username` reply-chain messages so chatters can respond to each other.
- Expanded neutral and idle message pools and added recent-message memory to reduce repeated lines in long combats.
- Centralized Fan threshold and chat intensity logic in a shared Livestream audience profile.
- Lowered the Livestream overlay canvas layer and clear it on combat victory so it should not appear above reward UI.

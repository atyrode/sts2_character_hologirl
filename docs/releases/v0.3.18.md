# Hologirl v0.3.18

- Fixed Livestream chat delivery by replacing the `_Process`-driven message queue with per-message Godot timers.
- Kept delayed reaction bursts, but made them flush even when the injected overlay node does not receive process callbacks.
- Moved message lifetime/fade handling onto per-message tweens so rows decay without relying on a frame loop.
- Refreshed ambient chat fan scaling immediately after Livestream generates Fans.

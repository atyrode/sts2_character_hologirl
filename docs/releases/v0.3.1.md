# Hologirl v0.3.1

Hotfix for a run-start black screen introduced in `v0.3.0`.

## Changes

- Kept Hologirl on BaseLib's `CustomCharacterModel`.
- Added explicit temporary Ironclad routing for in-run assets that Hologirl does not own yet: combat visual, run transition material and sound, energy counter, rest/merchant animations, hand textures, combat sounds, card trail, and temporary Architect attack VFX.
- Documented that these Ironclad routes are deliberate scaffolding, not passive `PlaceholderCharacterModel` inheritance.

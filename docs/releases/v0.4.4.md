# Hologirl v0.4.4

- Fixes the in-combat Hologirl creature scene export by making `Bounds` a `Control`, matching BaseLib's `NCreatureVisuals` auto-conversion expectations.
- Enlarges the custom in-combat energy counter.
- Rebalances Livestream chat so low fan counts produce fewer reactions and a smaller recurring chatter pool.
- Adds combat-victory chat reactions that can remain behind the reward screen.
- Tightens Livestream chat row-height estimation to avoid empty wrapped rows.
- Hardens the release workflow so GitHub releases can only publish committed source from `main` after it matches `origin/main`.

# Hologirl v0.3.5

- Added the first normal draftable Hologirl card pool so merchant and reward generation have valid non-basic cards.
- Added Broadcast Loop and nine temporary draftable cards across common, uncommon, and rare rarities.
- Marked Prism Pendant form reward cards as token cards so they stay out of normal merchant and reward pools.
- Removed Hologirl's custom in-run F3 debug panel now that DevMode covers manual run testing.
- Removed unused Harmony startup wiring and documented current library/tooling guidance.

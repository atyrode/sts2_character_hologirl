# Hologirl v0.3.19

- Added a larger stream-start greeting pool and made playing `Livestream` emit a small staggered opening burst.
- Increased Livestream chat message lifetime to about 10 seconds before fade-out.
- Expanded chat usernames with mixed casing, numbers, normal names, and mod-flavored handles.
- Rendered chat rows as one formatted rich-text line so messages consistently read as `username: message`.

# Hologirl v0.2.46

- Adds a `Body opacity` control to the F3 character-select tuner so Hologirl can be tuned more transparent like a hologram.
- Preserves gold-heavy whip/tool pixels while fading non-gold character pixels in the character-select shader.
- Adds five new humanoid hologram character-select cutout variants to the tuner.
- Archives the new generated cutout attempts with prompts and results under `docs/design/art_archive/menu/character_select_layers/hologirl_cutout/batch-2026-05-16-humanoid-hologram/`.
- Documents the updated cutout direction: humanoid projected hologram, thin selective outlines, cel-shaded/pastel grit, and less gem/slime read.

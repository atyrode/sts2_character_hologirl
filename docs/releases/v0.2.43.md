# Hologirl v0.2.43

- Adds five new thin-outline character-select cutout variants to the F3 tuner:
  - `Thin Outline Ready`
  - `Thin Outline Compact`
  - `Thin Outline Spire`
  - `Thin Outline Calm`
  - `Thin Outline Angular`
- Sets the current preferred character-select defaults:
  - `Periwinkle Drift` background
  - `White Gold Blue` character
  - subtle hologram tint, shimmer, and scanline values
  - tearing disabled by default
- Splits horizontal tearing into separate `Tear` and `Tear speed` tuner sliders.
- Archives the new cutout attempts and prompts in the design archive.

# Hologirl v0.3.22

- Increased Livestream inline emote display size and preserved their wider aspect ratio so they no longer render as tiny squished squares.
- Added partial chat pile-ons where a burst can include several variants of the same reaction family, such as laughter, confusion, hype, panic, or clean-play reactions.
- Expanded pile-on reaction pools with more casing, punctuation, short slang, and Hologirl emote usage.
- Kept pile-ons mixed with normal event-specific reactions so bursts remain less robotic while still sometimes feeling like chat collectively latched onto one moment.

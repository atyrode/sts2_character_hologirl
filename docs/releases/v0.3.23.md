# v0.3.23

- Temporarily adds `Livestream` to Hologirl's starter deck so the combat overlay and fan reactions can be tested immediately.
- Adds transparent, symbol-only power icons for `Livestream`, `Singing`, and `Fans`; generated sources are archived under `docs/design/art_archive/ui/power_icons/`.
- Expands Livestream chat with more usernames, mixed handle styles, multilingual recognizable phrases, emote-only posts, technical questions, serious comments, cute lines, haters, and longer chat-like reactions.
- Reparents the Livestream chat overlay to the creature visual layer so it should behave like character-space combat UI instead of sitting above pause/reward UI.
- Documents the power icon direction: simple readable symbols on transparency, with no colored backing plate or broad drop shadow.

# Hologirl v0.3.14

- Lowered the Livestream chat overlay layer so modal combat UI, such as potion targeting/menu UI, can render above it.
- Changed Livestream chat ordering so new messages appear from the bottom and older messages move upward.
- Tuned chat fade timing and kept messages capped to five visible rows.
- Changed chat formatting to `username: message` and tightened spacing between names and messages.
- Scaled hype-event reaction count with current `Fans`, with light random variance.

# v0.3.2

- Fixed `Livestream` so `Prism Pendant` owns and visibly triggers its Shapeshift reward-card effect.

- Added the remaining explicit Ironclad fallback icon routes from BaseLib's placeholder character shape, keeping Hologirl-owned menu art and audio separate.

# Hologirl v0.3.0

Release-candidate cleanup for Hologirl's first more stable playable direction.

## Changes

- Switched Hologirl to BaseLib's `CustomCharacterModel` path and removed placeholder character inheritance.
- Removed runtime character-select art variants from the shipped mod; archive/reference art remains in docs only.
- Locked character-select runtime art to the selected Periwinkle Drift background and Linework No Ref 02 cutout.
- Kept the F3 character-select tuner, but removed runtime background/cutout selectors.
- Applied the confirmed character-select hologram defaults.
- Replaced the stale locked character-select button art with the selected button art.
- Set Hologirl's card frame shader color from the canonical periwinkle character accent.
- Added simple effects for the four form reward cards instead of shipping no-op placeholder cards.
- Updated player-facing localization text away from template/prototype wording.
- Normalized local build/export tooling around the shared STS2, .NET, and Godot installations.

## Validation

- `scripts/build.sh`
- `scripts/godot-smoke-character-select.sh`
- `HOLOGIRL_PCK_EXPORTER=godot scripts/package.sh`

# Hologirl v0.2.24

- Keeps temporary character-select tuner values while switching characters, until the game process is restarted.
- Widens tuner ranges for more drastic calibration:
  - X: `0..2200`
  - Y: `-400..900`
  - Scale: `0.25..2.50`
  - Particle densities: `0..5`
  - Gold jitter: `0..80`
- Expands the Godot character-select smoke test to verify tuner state survives scene recreation.

# v0.4.1

- Rebalances `Fans` so they decay by half at end of turn unless Hologirl is `Singing`.
- Reworks the existing draftable pool around Fan spending instead of only gaining setup powers.
- Keeps `Livestream` as the signature rare power by removing the overlapping Livestream generation from `Prime Time`.
- Adds card-specific Livestream chat reactions for the reworked draftable cards and a recurring `Broadcast Loop` audience pulse.
- Makes `Viral Moment` require the full Fan threshold before spending, avoiding failed partial spends.
- Updates the card-art pass toward the flatter hand-drawn Hologirl direction, preserving the current white/gold outfit identity instead of the all-blue hologram look.
- Documents `Shapeshift`, Idol Forms, and form reward cards as deferred prototype mechanics for now.

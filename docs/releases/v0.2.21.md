# Hologirl v0.2.21

- Adds a temporary character-select tuning panel for X, Y, scale, whip particle density, hologram drift density, and gold jitter.
- The tuning panel can print and copy the current values so they can be applied as final constants after in-game testing.
- Reduces the default character-select particle burst and ongoing gold sparkle density.
- Tightens whip sparkle jitter and makes gold sparkles smaller so the effect is easier to align with the whip.

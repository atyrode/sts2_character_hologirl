# Layered Puppet Attempt 001

Generated on 2026-05-15 as a one-shot test for whether image generation can produce animation-ready Hologirl parts directly.

Result: useful as a concept sheet, not immediately production-ready. It separates major limbs, ponytails, torso, skirt, legs, whip, and sparkles, but still needs manual cleanup before rigging: consistent pivots, transparent extraction, exact scale matching, overlap margins, and likely redrawing of several joint cuts.

Keep this attempt archived as evidence for deciding whether to continue with a true layered-art workflow.

# Character Cutout Batch: Clean Hologram

Generated on 2026-05-15 as exploratory cutout attempts.

Goal: test whether Hologirl should remove baked scanlines from the source PNG and rely on procedural in-game hologram effects instead. These attempts improve the black outline/internal outline language, but skew too monochrome-blue compared to the later white/gold palette batch.

Selection status: archived for reference, not added to the runtime tuner.

## Prompt Direction

- Expressionless eyes-only Hologirl.
- Two long twin ponytails.
- Golden light-whip.
- Flat green chroma-key background.
- No baked scanlines or glitch bars.
- Stronger STS2-like black outlines.
- More solid rendered blue body than transparent glass.

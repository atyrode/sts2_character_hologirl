# Character Cutout Batch: White/Gold Palette

Generated on 2026-05-15 as runtime tuner variants.

Goal: explore Hologirl with visible colored clothes while preserving a holographic read. The strongest direction is white/gold outfit accents that echo the golden light-whip, with blue-tinted projected skin/hair and procedural in-game shimmer/tearing rather than baked scanlines.

## Prompt Direction

- White/gold outfit accents are allowed and encouraged.
- Hair/body can vary through blue, pink/lavender, cyan, or deep blue, but should retain a holographic blue cast.
- Strong black outer and internal outlines are required to better match vanilla STS2 character art.
- Source PNGs should not bake scanlines or tearing; the character-select shader adds subtle procedural hologram tint, shimmer, and horizontal tearing for these variants.

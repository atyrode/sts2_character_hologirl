Chroma key requirement: Complete subject on perfectly flat solid #00ff00 chroma-key background. Uniform color only, no shadow, gradient, texture, floor, or text.
Avoid: clean anime, beautiful detailed face, eyebrows, mouth, smile, anger, blush, detailed costume, photorealism, glossy highlights, smooth gradients, busy sparkles, many colors, extra characters, chibi proportions, cropped head, cropped ponytails, cropped whip, any green inside subject.

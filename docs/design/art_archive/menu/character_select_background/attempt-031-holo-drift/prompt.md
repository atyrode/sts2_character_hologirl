# Character Select Background Attempt 031: Holo Drift

Use case: stylized-concept

Asset type: Slay the Spire 2-inspired character-select background plate for the Hologirl mod. Wide horizontal game UI background, no character.

Primary request: Create a simple vanilla-feeling character-select background plate named Holo Drift. It should be a bright blue-violet solid accent field with only two close same-hue variants: darker indigo-violet silhouettes and lighter blue-violet highlights. The motif is large hand-painted hologram interference ribbons, like drifting scanline tears and broad signal distortion shapes, entering from offscreen and cropped.

Composition: wide 2564x1204-style plate. Leave the center-left readable for UI. Use one large darker irregular ribbon sweeping from lower left to upper middle, one cropped darker ribbon at the right edge, and a few sparse horizontal broken highlight strokes. Shapes should be broad, asymmetrical, hand-painted, and uneven. No small busy glitch texture.

Style direction: inspired by vanilla Slay the Spire 2 character-select backgrounds: low color count, flat matte color masses, broad symbolic motifs, hand-painted uneven silhouettes, simple graphic readability. No detailed environment, no character, no texture-heavy painting.

Color rules: one accent family only: blue-violet base, darker indigo-violet motifs, sparse lighter blue-violet highlights. Do not use pink, cyan, gold, orange, white glow, multicolor gradients, photoreal lighting, or black outlines.

Negative prompt: geometric vector art, clean SVG, perfect rectangles, digital glitch grid, tiny noise, centered logo, wallpaper pattern, detailed scene, texture, grain, smoky effects, realistic lighting, glossy glow, many colors, character, UI text, logo, watermark.

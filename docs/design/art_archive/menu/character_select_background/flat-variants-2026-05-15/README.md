# Character Select Backgrounds - Flat Variant Batch

Generated on 2026-05-15 after rejecting the previous generated background batches.

Direction:

- Bright solid accent-color base.
- One darker same-hue filled motif.
- Optional lighter same-hue outline or highlight.
- Two to three value variants total.
- No texture, grain, gradients, painterly lighting, soft shadows, scenery, or multicolor palettes.
- The motif should be simple but drawn: lightning, fire, curtains, signal waves, halo shapes.
- This background style is intentionally flatter and more cartoon-like than Hologirl's character layer.

Variants:

- `01-cyan-lightning.png`: cyan plate with darker cyan lightning motifs.
- `02-cobalt-flame.png`: cobalt plate with darker blue flame motifs.
- `03-rose-curtain.png`: rose plate with darker rose curtain motifs.
- `04-mint-signal.png`: mint plate with darker signal-wave and base motifs.
- `05-gold-halo.png`: gold plate with darker halo and ground motifs.

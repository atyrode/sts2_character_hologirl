# Character Select Backgrounds - Simple Variant Batch

Generated on 2026-05-15 as the second bitmap background exploration batch for the Hologirl character-select screen.

The first generated batch is kept for reference, but feedback was that it had the right abstraction level while still using too many details and color families. This batch intentionally pushes closer to vanilla character-select background plates: broad hand-drawn shapes, one accent hue family, two or three value variants, dark outlines, low detail, and large quiet fields behind the character.

Shared prompt constraints:

- Character-select background plate only; no character, no UI, no text.
- One accent hue family per image, with only lighter/darker variants of that hue.
- Two or three colors total in practice: near-black base, darker accent fill, muted lighter accent.
- Broad flat fills with thick dark edge definition.
- Slight crayon/pastel matte texture and minimal grain.
- No particles, stars, detailed scenery, multicolor gradients, or clean vector geometry.
- The character must remain the main focus.

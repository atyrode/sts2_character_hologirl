# Character Select Background Attempt 029: Signal Bloom

Use case: stylized-concept

Asset type: Slay the Spire 2-inspired character-select background plate for the Hologirl mod. Wide horizontal game UI background, no character.

Primary request: Create a simple vanilla-feeling character-select background plate named Signal Bloom. It should be a bright cyan-blue solid accent field with only two close same-hue variants: a darker cyan-blue and a paler cyan highlight. The motif is large hand-painted broadcast-wave silhouettes and soft signal arcs entering from offscreen, cropped by the image edges. It must feel hand-drawn and organic, not mathematical.

Composition: wide 2564x1204-style plate. Leave the center-left readable for UI. Put the largest darker motif mass around the far left edge and upper right edge, cropped offscreen. Use broad curved signal bands, uneven brush edges, and a few sparse tiny square signal flecks. No centered icon, no grid, no repeated pattern.

Style direction: inspired by vanilla Slay the Spire 2 character-select backgrounds: low color count, flat matte color masses, broad symbolic motifs, hand-painted uneven silhouettes, simple graphic readability. No detailed environment, no character, no texture-heavy painting.

Color rules: one accent family only: cyan-blue base, darker cyan-blue silhouettes, very sparse pale cyan highlights. Do not use pink, purple, orange, gold, white glow, multicolor gradients, photoreal lighting, or black outlines.

Negative prompt: geometric vector art, clean SVG, perfect circles, perfect arcs, radial symmetry, centered logo, wallpaper pattern, detailed scene, texture, grain, smoky effects, realistic lighting, glossy glow, many colors, character, UI text, logo, watermark.

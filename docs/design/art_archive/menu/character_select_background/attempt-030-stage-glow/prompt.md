# Character Select Background Attempt 030: Stage Glow

Use case: stylized-concept

Asset type: Slay the Spire 2-inspired character-select background plate for the Hologirl mod. Wide horizontal game UI background, no character.

Primary request: Create a simple vanilla-feeling character-select background plate named Stage Glow. It should be a bright desaturated rose-pink solid accent field with only two close same-hue variants: darker rose-plum silhouettes and pale pink highlights. The motif is large hand-painted stage-light cones and crowd/lightstick silhouette bands, abstracted so they read as an idol stage without becoming a scene.

Composition: wide 2564x1204-style plate. Leave the center-left readable for UI. Use one large darker spotlight cone entering from the upper left, another cropped darker cone/mass on the far right, and a low dark rounded band near the bottom edge suggesting distant crowd heads. Add very few tiny pale lightstick dashes, mostly near the bottom and edges. All shapes should be cropped, broad, organic, and uneven.

Style direction: inspired by vanilla Slay the Spire 2 character-select backgrounds: low color count, flat matte color masses, broad symbolic motifs, hand-painted uneven silhouettes, simple graphic readability. No detailed concert room, no character, no texture-heavy painting.

Color rules: one accent family only: rose-pink base, darker rose-plum motifs, sparse pale pink highlights. Do not use cyan, blue, gold, orange, white glow, multicolor gradients, photoreal lighting, or black outlines.

Negative prompt: geometric vector art, clean SVG, perfect cones, radial symmetry, literal stage scene, detailed crowd, many lightsticks, realistic lighting, smoky effects, texture, grain, many colors, character, UI text, logo, watermark.

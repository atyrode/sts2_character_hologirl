# Prompt

Use case: stylized-concept
Asset type: Slay the Spire 2 character-select button portrait for the Hologirl mod, vertical close-up icon that will be resized to 132x195.
Primary request: Create a small character-select portrait for Hologirl that matches the official STS2 character select portraits: tight bust/head crop, bold readable silhouette, simple colored background, hand-drawn matte fantasy style, no UI or text.
Composition: Vertical portrait. Hologirl fills most of the frame, head and shoulders visible, facing slightly left in a calm confident selection-pose. Two long ponytails/twintails must be clearly readable within the crop, with one or both ponytails entering the frame as large graphic shapes. Keep the face simple and stylized, not detailed. Leave enough silhouette contrast for the portrait to read at tiny size.
Subject: Hologirl is a blue/cyan hologram girl with headset-like performer earpieces, simple idol-like collar/bow shapes, transparent scanline-like hologram bands very subtle. She is not gold; her body and hair are blue/cyan. Add only tiny gold projected sparkles or a small gold light-whip curve near the edge if it helps identity.
Background: Flat, simple, saturated but not glossy desaturated magenta/plum background, similar simplicity to official STS2 char_select portraits. Minimal angular shadow shapes. No scene.
Style direction: Slay the Spire 2-inspired hand-drawn 2D fantasy portrait, chunky crisp shapes, strong outline/edge definition, matte brush texture, low detail, limited palette, readable at 132x195. Not anime-clean, not photorealistic, not glossy mobile-game art.
Avoid: text, logo, UI frame, full body, modern computer setup, detailed costume, detailed face, busy effects, crowd, many colors, chibi proportions, perfect anime lineart, photorealism.

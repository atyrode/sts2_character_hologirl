# Energy Symbol Attempt 002 Prompt

Use case: stylized-concept
Asset type: Hologirl energy symbol concept sheet for a Slay the Spire 2 character mod.
Primary request: Create one clean 4 columns x 3 rows concept sheet containing 12 distinct small energy-symbol icon variants only. No card frames, no card mockups, no UI panels, no labels, no readable text, no numbers, no logos.

Context: This symbol will replace the current generic gray energy orb for Hologirl. The vanilla card frame color will simply be periwinkle; this request is only for the character's custom energy symbol. The icon must feel compatible beside vanilla Slay the Spire 2 character energy symbols: compact, readable at tiny size, hand-drawn fantasy UI, simple silhouette, not a detailed illustration.

Hologirl identity: blue/cyan hologram girl, periwinkle character accent, projected golden light tools, stage/performance, broadcast signal, prism/hologram technology, light-whip.

Variant themes to explore across the 12 icons:

1. faceted periwinkle prism with tiny gold glint
2. hologram projector lens with broadcast arcs
3. small stage spotlight aperture
4. golden light-whip loop around periwinkle core
5. crystalline diamond with pixel shimmer
6. abstract twin-tail silhouette as an energy mark, very subtle
7. periwinkle starburst inside a dark rim
8. crescent projection arc over a cyan core
9. circular signal ring with one broken pixel edge
10. tiny hologram fan/crowd signal mark
11. simple faceted orb with diagonal shine
12. angular prism-heart/star hybrid, not cute, still STS-like

Style: Slay the Spire 2 inspired UI icon art. Matte flat color masses, chunky readable silhouettes, hand-drawn uneven edges, crisp dark outline or strong edge definition, restrained brush/pastel texture, selective hard shadow planes. Low color count. Periwinkle is the main color family. Cyan/blue can appear as hologram light. Gold/yellow only as a small projected-light accent.

Technical: Put each icon on a plain dark neutral background tile for comparison. Each icon should be centered and large enough to inspect, but designed to remain readable when scaled down to energy-icon size. Keep the icon shapes simple and not too thin.

Avoid: card frames, full cards, UI mockups, readable text, labels, numbers, logos, photorealism, glossy mobile-game orb, detailed fantasy illustration, anime-clean vector polish, excessive glow, many colors, tiny illegible linework, complex symbols.

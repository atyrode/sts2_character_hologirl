# Base Hologirl

## Confirmed Visual Direction

- Blue/cyan hologram girl.
- Two long ponytails/twintails, readable when hair is visible.
- Gold/yellow is not her body color; it is reserved for created/projected tools and effects.
- Base weapon is a gold/yellow holographic light-whip.

## Confirmed Gameplay Direction

Base Hologirl builds Fans, uses basic Strike/Defend tools, and can Shapeshift into Idol Forms.

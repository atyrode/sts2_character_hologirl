# Broadcast Loop

## Role

Common scaling power for steady audience growth.

## Current Mechanics

- Cost: 1
- Rarity: Common
- Type: Power
- At the start of each turn, gain 2 `Fans`.
- Upgrade: increases the per-turn `Fans` gain by 1.

## Design Notes

The card should feel like setting up a recurring stream schedule. It is pure audience infrastructure; payoff cards now spend the Fans it creates.

# Viral Moment

## Role

Rare tempo payoff for a built audience.

## Current Mechanics

- Cost: 2
- Rarity: Rare
- Type: Skill
- Spend 6 `Fans`.
- Gain 2 energy.
- Draw 2 cards.
- Gain 2 `Singing`.
- Upgrade: draw 1 additional card.

## Design Notes

This card converts audience into a highlight-turn refill. It is intentionally blank without enough Fans, so the player has to protect or build an audience before going viral.

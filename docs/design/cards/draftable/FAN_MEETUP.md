# Fan Meetup

## Role

Common defensive audience card.

## Current Mechanics

- Cost: 1
- Rarity: Common
- Type: Skill
- Spend up to 3 `Fans`.
- Gain 5 `Block`, plus 3 `Block` for each `Fan` spent.
- If no `Fans` were spent, gain 2 `Fans`.
- Upgrade: +2 `Block`.

## Design Notes

This card gives the common pool a defensive Fan outlet while still functioning as a starter audience-builder when Hologirl has no Fans yet.
